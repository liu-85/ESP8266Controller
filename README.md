# RC遥控器 (ESP8266 控制器)

这是一个基于 Android 的遥控器应用，专门设计用于控制运行特定固件的 ESP8266 设备。

## 主要功能

- **双摇杆控制**：左摇杆控制油门，右摇杆控制转向。
- **差速转向支持**：自动将油门和转向指令转换为左右电机速度（SS2 协议）。
- **舵机控制**：支持左转、回中、右转快捷按键。
- **自定义开关**：提供 4 个可自定义指令的开关（如灯光、蜂鸣器等）。
- **陀螺仪模式**：支持通过手机姿态感应进行控制。
- **连接方式**：支持 Wi-Fi (TCP) 和蓝牙连接。

## 硬件端配置 (ESP8266)

### 引脚定义
- **电机 1 (左轮)**: GPIO5 (D1), GPIO4 (D2)
- **电机 2 (右轮)**: GPIO14 (D5), GPIO12 (D6)
- **舵机**: GPIO13 (D7)

### 通信协议
应用默认使用 TCP 端口 `2000` 进行通信。

- **控制指令格式**: `SS2:油门值,转向值\n` (值范围 1000-2000)
- **舵机指令**:
  - 左转: `SERVO_LEFT\n`
  - 回中: `SERVO_CENTER\n`
  - 右转: `SERVO_RIGHT\n`
- **自定义开关**: 发送配置的自定义字符串指令（需在 App 设置中匹配）。

### 固件功能
ESP8266 端建议开启以下功能：
- **Web Server (Port 80)**: 用于网页端直接控制和 OTA 升级。
- **TCP Server (Port 2000)**: 接收来自此 App 的实时控制指令。
- **OTA 升级**: 支持通过 `/update` 路径进行无线固件更新。

## 使用说明

1. 确保手机与 ESP8266 连接到同一个 Wi-Fi 网络（例如 SSID: `xiaomi2`）。
2. 在 App 设置中输入 ESP8266 的 IP 地址（默认端口 2000）。
3. 点击“连接”即可开始控制。
4. 如果需要使用陀螺仪模式，请在主界面开启开关并点击“校准”。
